Midless is a free and open-source voxel game made in C.

*Multiplayer not fully functional yet*

## Controls

| Input                        | Action                |
|-------------------------------|----------------------|
| W A S D             | Move                           |
| Space               | Jump                           |
| Left Click          | Break block                    |
| Right Click         | Place block                    |
| Mouse wheel         | Block Selection                |
| T                   | Open Chat (Multiplayer Only)   |
| ESC                 | Open menu                      |

## Dependencies

| Dependency    | Version |
